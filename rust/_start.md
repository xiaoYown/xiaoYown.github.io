### 工具

- _rustup_: rustup 是 Rust 的安装和管理工具，并且官网推荐使用 rustup 安装 [Rust](https://www.rust-lang.org/tools/install)
- _Cargo_: Cargo 是一个工具,允许 Rust 项目声明其各种依赖项，并确保您始终获得可重复的构建
